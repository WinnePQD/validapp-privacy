Política de Privacidade — ValidApp

Última atualização: 09/12/2025

O ValidApp respeita sua privacidade e está comprometido com a proteção dos seus dados.

Ao utilizar o ValidApp, você concorda com os termos desta política.

1. Informações Coletadas

1.1 Dados fornecidos pelo usuário

  • Informações de produtos cadastrados (nome, validade, categoria etc.)
  • Fotos tiradas pelo usuário dos produtos

1.2 Uso da câmera

O app solicita acesso à câmera para leitura de códigos de barras e captura de imagens. Nenhuma imagem é enviada para servidores externos sem ação do usuário.

1.3 Imagens baixadas da internet

O ValidApp busca automaticamente imagens públicas de produtos na internet com base no código de barras. As imagens são armazenadas apenas no dispositivo.

1.4 Dados de login

O usuário pode criar login e senha. Esses dados são armazenados localmente e no Firebase (Authentication e Realtime Database) para sincronização segura.

2. Uso das Informações

Os dados são usados exclusivamente para autenticação, cadastro de produtos, leitura de códigos de barras e sincronização da conta.

3. Armazenamento e Segurança

O app utiliza armazenamento interno do dispositivo e Firebase, com regras de segurança e proteção por senha.

4. Compartilhamento de Dados

O ValidApp não compartilha dados pessoais com terceiros. Utiliza apenas Firebase Authentication e Realtime Database.

5. Retenção e Exclusão

O usuário pode solicitar exclusão da conta. Dados locais podem ser apagados desinstalando o aplicativo.

6. Crianças

O app não é destinado a menores de 18 anos.

7. Permissões

  • Câmera (android.permission.CAMERA)
  • Armazenamento interno do aplicativo

8. Alterações

Esta política pode ser atualizada periodicamente.
